# Teslo API

1. Clone this repository
2. ```` yarn start````
3. Clone the file ```` .env.template```` and rename it to ```` .env````
4. Change the environment variables
5. Lift the database
 ````
 docker-compose up -d
 ````

6. Lift 
 ````
 yarn start:dev
 ````
